# 賓果預測器 - 用戶快速開始指南

## 🎯 5 分鐘快速部署

本指南適合所有用戶，無需技術背景。

---

## 📋 前置準備

### 您需要準備的東西

1. **中央數據庫連接字符串** ← 從原始帳號獲取
2. **電腦上已安裝**：
   - Node.js（[下載](https://nodejs.org)）
   - Git（[下載](https://git-scm.com)）

### 檢查是否已安裝

打開終端/命令提示符，執行以下命令：

```bash
node --version
# 應該顯示 v16.0.0 或更高版本

git --version
# 應該顯示 git version ...
```

如果顯示「command not found」，請先安裝相應軟件。

## 🚀 部署步驟

### 步驟 1：打開終端/命令提示符

**Windows 用戶：**
- 按 `Win + R`
- 輸入 `cmd` 並按 Enter

**Mac 用戶：**
- 按 `Cmd + Space`
- 輸入 `terminal` 並按 Enter

**Linux 用戶：**
- 按 `Ctrl + Alt + T`

### 步驟 2：下載應用代碼

複製以下命令到終端，然後按 Enter：

```bash
git clone https://github.com/2297gupy-cloud/bingo-predictor.git
cd bingo-predictor
```

**您會看到：**
```
Cloning into 'bingo-predictor'...
remote: Enumerating objects: 150, done.
...
```

### 步驟 3：執行部署腳本

#### Windows 用戶

```bash
deploy.sh
```

#### Mac 和 Linux 用戶

```bash
chmod +x deploy.sh
./deploy.sh
```

## ▶️ 啟動應用

部署完成後，執行以下命令啟動應用：

```bash
cd bingo-predictor
pnpm dev
```

**您會看到：**
```
  VITE v5.0.0  ready in 234 ms

  ➜  Local:   http://localhost:3000/
  ➜  press h to show help
```

### 打開應用

在瀏覽器中打開：
```
http://localhost:3000
```

**應用應該會顯示賓果預測器的主頁。**

## 🔄 驗證共享數據庫

### 測試數據同步

1. **在應用中添加一個投注**
   - 選擇星級（如 5 星）
   - 選擇倍數（如 x3）
   - 選擇大小或單雙
   - 點擊「加入投注」

2. **檢查結果頁面**
   - 點擊「結果」標籤
   - 應該能看到剛才添加的投注記錄
   - 驗證金額、倍數、期數都正確

3. **在另一個應用中查看**
   - 如果有其他用戶也部署了此應用
   - 他們應該能在自己的結果頁面看到您的投注記錄
   - 這證明數據庫同步成功！

## ❌ 常見問題排查

### 問題 1：「command not found: node」

**解決方案：**
- Node.js 未安裝
- 從 https://nodejs.org 下載並安裝
- 重新打開終端
- 執行 `node --version` 驗證

### 問題 2：「無法連接到數據庫」

**解決方案：**
1. 檢查 DATABASE_URL 是否正確複製
2. 確保數據庫伺服器正在運行
3. 檢查網絡連接
4. 嘗試重新執行 `pnpm db:push`

### 問題 3：「端口 3000 已被佔用」

**解決方案：**
```bash
# 使用不同的端口
PORT=3001 pnpm dev
```

然後在瀏覽器中打開 `http://localhost:3001`

### 問題 4：「依賴安裝失敗」

**解決方案：**
```bash
# 清除緩存
pnpm store prune

# 刪除舊文件
rm -rf node_modules pnpm-lock.yaml

# 重新安裝
pnpm install
```

### 問題 5：「應用無法啟動」

**解決方案：**
1. 檢查終端中的錯誤信息
2. 確保 DATABASE_URL 已正確設置
3. 執行 `pnpm db:push` 同步數據庫
4. 重新啟動應用 `pnpm dev`
